## The work behind the work

Since March 2025, I have been doing hands-on AI product engineering for early-stage startups: PDF data extraction, SEO analytics, a GenAI dermatopathology portal, AI chat/GPT experiences, smart scraping agents, and agent harnesses, skills, and automations.

Before that, I specialised in data-rich product interfaces:

- At **Hunt Intelligence**, I built threat-intelligence visualisation components, product modules including AttackCapture™ and HuntSQL™, and an OpenAPI-based API documentation experience.
- At **OMEGA Systems**, I helped build CORE5, visualised product data, and later led frontend onboarding and async delivery practices.
- At **Phantasma Chain** and **BinaryEdge / Coalition**, I built and led frontend systems, explorers, component libraries, and customer-security products.

That path has made me comfortable taking a product from a fuzzy brief through architecture, design systems, CI/CD, testing, and a team’s day-to-day way of shipping.
