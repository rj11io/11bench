## Why I care about this work

I started coding young by modding and reverse-engineering games and consoles, building a fighting game on MUGEN, and running servers for Counter-Strike and Minecraft. That curiosity eventually turned into a career in web product engineering.

Two things still shape how I work:

1. I like systems that are technically solid and easy for others to pick up.
2. I like interfaces that make complex data feel understandable instead of intimidating.
