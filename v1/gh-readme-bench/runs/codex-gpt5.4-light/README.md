# Ricardo Jorge

AI Product Engineer based in Lisbon, building the layer where AI systems,
product UX, and real-world data workflows meet.

I started out modding games, reverse-engineering consoles, and building things
for fun. These days I help turn early-stage ideas into usable products: AI
workflows, dashboards, data explorers, agent harnesses, and the frontend
systems that make them feel polished instead of provisional.

## What I Build

- **AI product work**: PDF data extraction, AI SEO analytics, GenAI product
  experiences, agent harnesses, custom skills, and n8n automations.
- **Data-heavy interfaces**: cybersecurity dashboards, proprietary data
  explorers, reporting surfaces, and visualization-driven product modules.
- **Foundations that scale**: TypeScript architecture, React/Next.js apps,
  component libraries, testing pipelines, and onboarding systems for teams that
  are still forming.
